# API
#api 

## Co to jest?

**API** - Application Programming Interface. 
To kontrakt, który umożliwia komunikację pomiędzy jednym serwisem/aplikacją a drugim serwisem/aplikacją

## API contract

| Request    | ->  | Response |
| ---------- | --- | -------- |
| - endpoint (URL) |     | - status code         |
| - headers  |     | - headers         |
| - body           |     | - body         |
