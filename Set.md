#java 

```java
Set<String> fruits = new HashSet<>();
```

Dodawanie elementów
```java
fruits.add("apple");  
fruits.add("banana");  
fruits.add("lemon");  
fruits.add("orange");
```

Usuwanie elementów
```java
fruits.remove("lemon");
```

Sprawdzenie, czy dany element istnieje
```java
System.out.println(fruits.contains("kiwi"));
```

Sprawdzenie rozmiaru kolekcji
```java
System.out.println(fruits.size());
```
